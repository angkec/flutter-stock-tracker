# 盯喵 - 应用品牌设计

## 应用名称

**中文名：盯喵**

- 谐音"盯"（盯盘）+ "喵"（猫叫声）
- 简洁好记，两个字
- 可爱又不失功能表达
- 英文/拼音：DingMiao

## 图标设计

### 核心元素

猫眼特写

### 设计规格

- **主体**：一只圆润的猫眼占据整个图标
- **虹膜**：金色/琥珀色，带渐变光泽
- **瞳孔**：黑色竖瞳，内嵌简化 K 线图（2-3 根红绿蜡烛）
- **眼眶**：深色/黑色描边，简约利落
- **背景**：透明或深色（适配深色模式）

### 视觉效果

- 远看：一只金色猫眼，辨识度高
- 近看：瞳孔里藏着股市 K 线，体现"盯盘"

### 配色

| 元素 | 颜色 |
|------|------|
| 虹膜 | 金色/琥珀色渐变 (#FFD700 → #FF8C00) |
| 瞳孔 | 黑色 (#1A1A1A) |
| K线-涨 | 红色 (#FF4444) |
| K线-跌 | 绿色 (#00AA00) |
| 眼眶 | 深灰/黑色 (#333333) |

### 尺寸要求

- iOS: 1024x1024 (App Store), 180x180 (@3x)
- Android: 512x512 (Play Store), 192x192 (xxxhdpi)
- macOS: 1024x1024, 512x512, 256x256, 128x128, 64x64, 32x32, 16x16
