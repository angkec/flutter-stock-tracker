# 行业异常买入指标说明（Z / 广度 / Q 对照表）

本页面用于 **UI 直接展示 + 用户理解**。  
所有指标均来自「分钟级量价协同 + 行业聚合」模型，**不是价格指标，也不是择时信号**。

---

## 一、单项指标释义对照表

### 1️⃣ Z 值（Z_rel）：异常买入强度

**Z 值回答的问题：**
> 今天这个行业，相比它自己过去 + 相比全市场，  
> **是不是出现了异常的主动买入倾向？**

| Z 值区间 | 语义解释 | 行为层面含义 |
|---|---|---|
| `Z < 0` | 无异常 | 卖出或弱于市场 |
| `0 ~ 0.5` | 正常波动 | 没有值得注意的资金行为 |
| `0.5 ~ 1.0` | 轻微偏买 | 可能是噪音 |
| `1.0 ~ 1.5` | 明显偏买 | **开始不寻常，值得关注** |
| `1.5 ~ 2.0` | 强异常 | **高度疑似资金在主动配置** |
| `> 2.0` | 极端异常 | 事件 / 集体行为 / 建仓加速 |

⚠️ 说明  
- **Z 高 ≠ 价格马上上涨**  
- Z 只说明：**资金行为“不正常”**，不说明阶段位置  

---

### 2️⃣ 广度（Breadth）：行业一致性

**广度回答的问题：**
> 是“整个行业在动”，  
> 还是“只有少数几只股票在动”？

定义：
```

广度 = 行业内，买入倾向为正的股票数 / 通过过滤的股票总数

```

| 广度区间 | 行业结构含义 |
|---|---|
| `< 0.25` | 极度集中（1~2 只票） |
| `0.25 ~ 0.35` | 初步扩散 |
| `0.35 ~ 0.45` | **典型行业级行为** |
| `0.45 ~ 0.60` | 强一致性 |
| `> 0.60` | 情绪 / 趋势阶段常见 |

⚠️ 说明  
- **早期建仓时，广度往往偏低**  
- 广度高，说明“面”已经铺开，但也可能更接近中后段  

---

### 3️⃣ Q 值（Quality）：信号可信度

**Q 回答的问题：**
> 当前的 Z 和广度，  
> **在统计结构和行为结构上，值不值得信？**

Q 是多个过滤条件的合成评分，包括：
- 成份股数量是否足够
- 是否被少数股票劫持
- 广度是否合理
- 是否具有一定持续性

| Q 值区间 | 可信度解释 |
|---|---|
| `< 0.40` | 噪音为主，不建议采信 |
| `0.40 ~ 0.60` | 勉强可看，观察级 |
| `0.60 ~ 0.75` | **可信信号区** |
| `> 0.75` | 结构非常干净，罕见高质量 |

⚠️ 说明  
- **Q 高 ≠ 一定要操作**  
- Q 高 = “你看到的不是统计幻觉”

## 二、三指标组合语义对照表（核心）

### 🟢 状态 A：早期疑似建仓（Early Accumulation）

| 指标 | 典型范围 |
|---|---|
| Z | `1.2 ~ 1.8` |
| 广度 | `0.25 ~ 0.40` |
| Q | `> 0.60` |

**含义：**
- 有明显异常买入
- 但只集中在部分核心股票
- 行业尚未全面启动

**UI 标签建议：**
> 🟢 早期建仓（观察）

---

### 🟡 状态 B：行业性配置（Broad Accumulation）

| 指标 | 典型范围 |
|---|---|
| Z | `> 1.5` |
| 广度 | `0.40 ~ 0.55` |
| Q | `> 0.65` |

**含义：**
- 行业内多数股票出现买入倾向
- 非单点行为
- 很像机构/主力在做行业级配置

**UI 标签建议：**
> 🟡 行业配置期

---

### 🔴 状态 C：情绪 / 事件驱动（Momentum / Hype）

| 指标 | 典型范围 |
|---|---|
| Z | `> 2.0` |
| 广度 | `> 0.55` |
| Q | 不稳定（可能高也可能下降） |

**含义：**
- 全行业高度一致买入
- 往往伴随新闻、主题、政策、情绪
- 不一定是“建仓”，可能是博弈或兑现阶段

**UI 标签建议：**
> 🔴 情绪驱动（注意风险）

---

### ⚠️ 状态 D：可疑信号（Low Quality Spike）

| 指标 | 常见特征 |
|---|---|
| Z | 偏高 |
| 广度 | 很低（< 0.25） |
| Q | 低（< 0.50） |

**含义：**
- 极可能被少数股票或异常分钟劫持
- 或数据结构不干净
- 不具备行业代表性

**UI 标签建议：**
> ⚠️ 噪音信号

---

### 🔵 状态 E：无显著行为（Neutral）

| 指标 | 范围 |
|---|---|
| Z | `-0.5 ~ 0.5` |
| 广度 | 任意 |
| Q | 任意 |

**含义：**
- 行业行为与历史无显著差异
- 无需特别关注

**UI 标签建议：**
> ⚪ 无异常

---

## 三、UI 展示建议（可直接用）

### 1️⃣ 单行业卡片推荐字段

- 行业名称
- `Z`（数值 + 颜色梯度）
- `广度`（百分比）
- `Q`（0~1）
- 状态标签（由三指标组合映射）

### 2️⃣ 排序推荐

默认排序：
```

按 Z 值降序
→ 再按 Q 值降序

```

### 3️⃣ 快速筛选建议

- `Z > 1.2`
- `Q > 0.6`
- `广度 > 0.3`

---

## 四、一句话给用户看的解释（可放 tooltip）

- **Z 值**：这个行业今天“买得是否不正常”
- **广度**：是不是整个行业在一起买
- **Q 值**：这个信号靠不靠谱

---

## 五、重要免责声明（建议 UI 中弱提示）

> 本指标刻画的是 **资金行为异常**，  
> 不代表价格走势预测，  
> 也不构成任何投资建议。
